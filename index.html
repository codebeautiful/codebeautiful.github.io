命令模式下，d(D) 操作总结

  1. 删除当前光标下的整个单词： daw
  2. 删除当前光标到单词首：db
  3. 删除当前光标到单词尾：dw
  4. 删除当前光标所在位置到行首：d^ | d0(包括空格，tab)
  5. 删除当前光标到行尾：d$ | D
  6. 删除到下一句的开始：d)
  7. 删除到下一段的开始：d}
  8. 删除从文本中出现 “text” 字符的位置：d/text
    9.

移动命令（非常用）

  1. 光标移动屏幕顶行 H
  2. 光标移动屏幕中央 M
  3. 光标移动屏幕底部 L
  4. 查询当前行信息和当前文件信息 C-g
  5. 将第 ＃ 行滚至屏幕顶部 #z

进入和退出 vim 命令

进入命令

  1. 打开文件，光标置于第一行 vim filename
  2. 打开文件，光标置于 n 行 vim +n filename
  3. 打开文件，光标置于最后一行 vim + filename
  4. 打开文件，将光标置于与 pattern 第一个匹配的地方vim +/pattern filename
  5. 打开文件，并恢复上次崩溃的文件 vim -r filename
  6. 打开文件前，先执行指定的命令 vim -c cmd file

-R 以只读的方式打开文件，但可强制保存
-M 以只读方式打开文件，但是不可强制保存
vim –remote file : 用已有的vim进程打开指定的文件

退出命令

  1. ZZ
  2. :q!
  3. :wq

文档操作

  1. :e file –关闭当前编辑的文件，并开启新的文件。 如果对当前文件的修改未保存，vi会警告。
  2. :e! file –放弃对当前文件的修改，编辑新的文件。
  3. :e+file – 开始新的文件，并从文件尾开始编辑。
  4. :e+n file – 开始新的文件，并从第n行开始编辑。
  5. :enew –编译一个未命名的新文档。(CTRL-W n)
  6. :e – 重新加载当前文档。
  7. :e! – 重新加载当前文档，并丢弃已做的改动。
  8. :e#或ctrl+^ – 回到刚才编辑的文件，很实用。
  9. :f或ctrl+g – 显示文档名，是否修改，和光标位置。
  10. :f filename – 改变编辑的文件名，这时再保存相当于另存为。
  11. gf – 打开以光标所在字符串为文件名的文件。
  12. :w – 保存修改。
  13. :n1,n2w filename – 选择性保存从某n1行到另n2行的内容。
  14. :wq – 保存并退出。
  15. ZZ – 保存并退出。
  16. :x – 保存并退出。
  17. :q[uit] ——退出当前窗口。(CTRL-W q或CTRL-W CTRL-Q)
  18. :saveas newfilename – 另存为
  19. :browse e – 会打开一个文件浏览器让你选择要编辑的文件。 如果是终端中，则会打开netrw的文件浏览窗口； 如果是gvim，则会打开一个图形界面的浏览窗口。 实际上:browse后可以跟任何编辑文档的命令，如sp等。 用browse打开的起始目录可以由browsedir来设置：
  20. :set browsedir=last – 用上次访问过的目录（默认）；
  21. :set browsedir=buffer – 用当前文件所在目录；
  22. :set browsedir=current – 用当前工作目录；
  23. :Sex – 水平分割一个窗口，浏览文件系统；
  24. :Vex – 垂直分割一个窗口，浏览文件系统；

特别的移动命令

    • / Enter : 把光标移动至下一行第一个非空白字符
    • 把光标移至上一行第一个非空白字符

g 命令使用总结

ge 后移一个单词，光标停在上一个单词末尾

gE 反向转移到上一个 WORD 的结尾处

gv 重新选择上次的可视区域

gd 转到当前函数里当前光标所在的单词的定义

gD 转到当前文件的光标所在单词的定义

gf 开始编辑当前光标所在的名字对应的文件

gF 开始编辑光标所在的文件名并跳转到文件名后的

gJ 连接行，不插入空格

ga 打印光标所在处的字符的 ascii 值

g8 显示光标所在的 UTF-8 字符的十六进制字节码

gg 光标移动到第 N 行，缺省是第一行

插入模式

  1. 进入替换模式，直到按下 esc R

^V 按下^V后即可进行矩形的选择

  1. aw 选择单词
  2. iw 内部单词(无空格)
  3. as 选择句子
  4. is 选择句子(无空格)
  5. ap 选择段落
  6. ip 选择段落(无空格)
  7. D 删除到行尾
  8. x,y 删除与复制包含高亮区
  9. I 编辑相同列，可添加注释

fx 向右跳到本行字符x处（x可以是任何字符）
Fx 向左跳到本行字符x处（x可以是任何字符）

标记

使用标记可以快速移动。到达标记后，可以用Ctrl+o返回原来的位置。 Ctrl+o和Ctrl+i 很像浏览器上的 后退 和 前进 。

m{a-z}: 标记光标所在位置，局部标记，只用于当前文件。
m{A-Z}: 标记光标所在位置，全局标记。标记之后，退出Vim， 重新启动，标记仍然有效。
{a-z}: 移动到标记位置。 '{a-z}: 移动到标记行的行首。{0-9}：回到上[2-10]次关闭vim时最后离开的位置。
: 移动到上次编辑的位置。''也可以，不过精确到列，而’’精确到行 。如果想跳转到更老的位置，可以按C-o，跳转到更新的位置用C-i。
": 移动到上次离开的地方。.: 移动到最后改动的地方。
:marks 显示所有标记。
:delmarks a b – 删除标记a和b。
:delmarks a-c – 删除标记a、b和c。
:delmarks a c-f – 删除标记a、c、d、e、f。
:delmarks! – 删除当前缓冲区的所有标记。
:help mark-motions 查看更多关于mark的知识。
