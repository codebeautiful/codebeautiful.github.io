Android 新建工程报错 Could not resolve com.android.tools.build:gradle:8.0.1.

7 May, 2023

解决方法修改Gradle JDK版本

File > Settings > Build, Execution, Deployment > build tools > gradle Then Change Gradle JDK from the dropdown and select Android Studio java home
