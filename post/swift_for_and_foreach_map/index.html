Switch使用for forEach map

20 April, 2023

区间用法

区间运算符 (… 和 ..<) ，分别表示

0 … 3 表示 [0,1,2,3]，数学表达式[0,3]

0 ..< 3 表示[0,1,2],数学表达式[0,3)

代码如下 

for i in 0 ... 3 {
  print(i)// 0,1,2,3
}
for i in 0 ..< 3 {
  print(i)// 0,1,2
}
// 倒序遍历
for i in (0...3).reversed() {
  print(i)// 3,2,1,0
}
// 条件遍历
for i in 0...3 where i % 2 == 0 {
  print(i)// 0,2
}

数组遍历方法 

let arr = [1,2,3]
// 元素遍历
for a in arr {
    print(a)
}
// 索引遍历
for i in 0..<arr.count {
    print(arr[i])
}
// forEach
arr.forEach { a in
    print(a)
}
// 迭代器遍历
for (index, element) in arr.enumerated() {
    print(index, element)// 0 1, 1 2, 2 3
}
